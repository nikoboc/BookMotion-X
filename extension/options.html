Kindle → Notion 設定

1. Notion インテグレーション

  1. notion.so/my-integrations で内部インテグレーションを作成し、Internal Integration Token をコピー
  2. DB を置きたい親ページを Notion で開き、右上「•••」→「連携」→ 作成したインテグレーションを追加（これを忘れると 404 になります）
  3. その親ページの URL 末尾の 32 桁 ID（または URL 丸ごと）を下に貼る
ここに DB を新規作成します。

2. データベース作成

指定の 6 列（ハイライト文 / 書籍名 / 著者名 / 位置 / マーカー色 / 実行日）で作成します。 作成済みなら不要です。
既存 DB を使う場合はここに直接 ID を入れて「保存」。列名は上記 6 つと一致させてください。
